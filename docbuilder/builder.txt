#IEX Stock Endpoint
balance_sheet(symbol, vprint=False, **queries)
Returns the balance sheet fincancial statement for the requested stock.
book(symbol, vprint=False)
Returns share book information for the requested stock.
cash_flow(symbol, vprint=False, **queries)
Returns the cash flow financial statement for the requested stock.
collection(collectionType, collectionName, vprint=False)
Returns quotes for stocks in the requested collection type.
company(symbol, vprint=False)
Returns company data such as website, address, and description for the requested stock.
delayed_quote(symbol, vprint=False)
Returns a 15-minute delayed market quote for the requested stock.
dividends(symbol, scope, vprint=False)
Returns dividend information for a requested stock.
earnings(symbol, last=None, field=None, vprint=False)
Returns earnings data such as actual EPS, beat/miss, and date for the requested stock.
estimates(symbol, vprint=False)
Returns latest future earnings estimates for the requested stock.
financials(symbol, period=None, vprint=False)
Returns a brief overview of a company's financial statements.
fund_ownership(symbol, vprint=False)
Returns the largest 10 fund owners of the requested stock. This excludes explicit buy or sell-side firms.
historical_price(symbol, period, date=None, vprint=False, **queries)
Returns the historical price for the requested stock.
income_statement(symbol, vprint=False, **queries)
Returns the income statment financial data for the requested stock.
insider_roster(symbol, vprint=False)
Returns the 10 largest insider owners for the requested stock.
insider_summary(symbol, vprint=False)
Returns a summary of the insiders and their actions within the last 6 months for the requested stock.
insider_transactions(symbol, vprint=False)
Returns a summary of insider transactions for the requested stock.
institutional_ownership(symbol, vprint=False)
Returns the 10 largest instituional owners for the requested stock. This is defined as explicitly buy or sell-side only.
ipo_today(vprint=False)
Returns a list of IPOs happening today.
ipo_upcoming(vprint=False)
Returns a list of upcoming IPOs for the current and next month.
key_stats(symbol, stat=False, vprint=False)
Returns important and key statistics for the requested stock.
largest_trades(symbol, vprint=False)
Returns a delayed list of largest trades for the requested stock.
logo(symbol, vprint=False)
Returns a Google APIs link to the logo for the requested stock.
market_list(symbol, list_type, displayPercent=None, vprint=False)
Returns the 10 largest companies in the specified list.
market_volume(format=None, vprint=False)
Returns market wide trading volume.
news(symbol, last=None, vprint=False)
Returns news item summaries for the requested stock.
ohlc(symbol, vprint=False)
Returns the most recent days open, high, low, and close data for the requested stock.
peers(symbol, vprint=False)
Returns a list of a requested stocks peers.
previous(symbol, vprint=False)
Returns the previous day's price data for the requested stock.
price(symbol, vprint=False)
Returns a single float value of the requested company's price.
price_target(symbol, vprint=False)
Returns analyst's price targets for the requested stock.
quote(symbol, field=None, vprint=False)
Returns price quote data for the requested stock. Fields are able to be called individually.
recommendation_trends(symbol, vprint=False)
Returns analyst recommendations for the requested stocks.
sector_performance(vprint=False)
Returns market performance for all sectors.
splits(symbol, scope=None, vprint=False)
Returns a record of stock splits for the given stock.
today_earnings(vprint=False)
Returns earnings data released today, grouped by timing and stock.
volume_by_venue(symbol, vprint=False)
Returns trading volume for the requested stock by venue.
#IEX Forex Endpoint
forex_conversion(symbols, amount, vprint=False)
Returns a converted value according to the requested currency pair.
forex_historical(symbols, vprint=False, **queries)
Returns a list of the historical exchange rates for the requested currency pair.
forex_latest_rate(symbols, vprint=False)
Returns the latest exchange rate for the requested currency pair.
#IEX Market Endpoint
commodities(symbol, vprint=False)
Returns commodites data for the requested commodities symbol.
economic_data(symbol, vprint=False)
Returns economic data for the requested economic indicator symbol.
generic_data_point(symbol, key=None, vprint=False)
Generic endpoint used to access all 'Data Point' data sets on IEX.
generic_time_series(symbol, *args, vprint=False)
Generic endpoint used to access all 'Time Series' data sets on IEX
# FRED API Endpoints
fred_series(series, file_type=None, realtime_start=None, realtime_end=None, limit=None, offset=None, sort_order=None, observation_start=None, observation_end=None, units=None, frequency=None, aggregation_method=None, output_type=None, vintage_dates=None)
Returns time series historical data for the requested FRED data.
geofred_regional_series(series_id, file_type=None, date=None, start_date=None)
Returns the historical, geographically organized time series data for the requested FRED data.
geofred_series_meta(series_id, file_type=None)
Returns meta data for the requested FRED data.
get_yield()
Returns government treasury bond yields. Organized in Python dictionary format by bond length.
# EDGAR Report Handling
accession_10k(self, count, get=False)
Returns accession numbers for all the 10-Ks for the desired company.
accession_10q(self, count, get=False)
Returns accession numbers for all the 10-Qs for the desired company.
accession_all_recent(self, count, get=False)
Returns all the most recent accession numbers for the desired company.
accession_requests(self, count, document)
Returns accession numbers for the requested document.
accessions(self, count, documents, get=False, html=False, xbrl=False, xlsx=False, strip=False)
Returns accession numbers and documents in five forms for all the documents for the desired company.
cik(self)
Sets the CIK attribute for the requested company.
retrieve_html_report(self, accession, save=False, open=False)
Strips non-html elements from the raw filing document.
retrieve_txt_report(self, accession, save=False, classify=False)
Saves the raw xml filing for the given accession number.
retrieve_xbrl_report(self, accession, save=False)
Opens a browser to the Inline XBRL Interactive report.
retrieve_xlsx_report(self, accession)
Streams financial statement Excel file, may only work on statements in the last 10 years.
# Auxiliary Functions
pick_bad_apples(word, ignore_list)
Determines whether or not to ignore certain words.
real_word_frequency(file)
Returns a list fo real words sorted by use frequency.
