About the Wrapper:
This is a minimal wrapper built to take advantage of various financial APIs. The idea is that with one package you can access all the data you might need for your financial application from one convenient place. Having multiple API's available through one package affords better clarity in programs by reducing imports and standardizing language used.

Original Documentation:
As many sources are coming together, we will try our best to link directly to original documentation pages and example JSON links when possible, using this page as a comprehensive list of functions rather than a "How To" guide.
